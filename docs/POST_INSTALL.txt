Post-install checklist â€” things you MUST change after extracting this ZIP:

1) Set your HuggingFace API key:
   - Create backend/.env and add:
     MONGO_URI=mongodb+srv://<user>:<pass>@cluster0.mongodb.net/recipes
     HF_API_KEY=hf_your_key_here
     PORT=5000

2) Install dependencies:
   - Backend:
     cd backend
     npm install
   - Frontend:
     cd frontend
     npm install

3) Seed MongoDB:
   - Make sure MONGO_URI is set in backend/.env
   - Run: npm run seed (from backend) to populate recipes

4) Start servers:
   - Backend: npm run dev (or npm start)
   - Frontend: npm run dev

5) Update API URLs in frontend if deploying (search for http://localhost:5000).

6) Tailwind CSS:
   - If you want to customize Tailwind, run: npx tailwindcss init and update config

7) Do NOT commit your .env file to Git.
